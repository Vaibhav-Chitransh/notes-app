# notes-app

A simple note-taking app where you can create, updated, delete and read your notes. So why I am making this simple application, to be frank I don't know it's just that I am bored and want to make something maybe that's the reason.

### Functional Requirements

- [ ] User Signup, login and logout
- [ ] User can create, read, update and delete the notes
- [ ] Note contains title, content, tags and color code
- [ ] Filtering notes by tags, date modified and color code
- [ ] Search bar for searching notes by title and content
- [ ] Light and dark mode
- [ ] Responsive design
- [ ] Pagination to display notes on screen
- [ ] Sort notes by date modified or aplabetical title


### Models

- User
- Note
- Tag
